# GeoExplorer | Country Information Discovery

GeoExplorer is a modern, sleek web application that allows users to discover detailed information about countries worldwide. Powered by the REST Countries API, it provides a beautiful interface with glassmorphism design and fluid interactions.

![GeoExplorer Preview](https://via.placeholder.com/800x450?text=GeoExplorer+Premium+Design)

## ‚ú® Features

- **Global Search**: Find any country by its common or official name.
- **Detailed Information**: View capital, population, region, currencies, languages, and timezones.
- **Interactive Borders**: Discover and navigate to neighboring countries with a single click.
- **Dynamic Backdrop**: The background elegantly transitions to match the searched country's flag.
- **Visual Assets**: High-quality SVG flags and official Coats of Arms.
- **Maps Integration**: One-click access to the country's location on Google Maps.
- **Responsive & Modern**: Fully responsive glassmorphism UI designed with a dark-mode first approach.
- **Fluid Experience**: Smooth animations, loaders, and transitions for a premium feel.
